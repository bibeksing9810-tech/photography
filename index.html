About Me

Hello, I'm Bibek Singh

I'm a professional photographer and full stack developer with a passion for capturing special moments and creating lasting memories for my clients.

My journey in photography began during my travels around Nepal, where I discovered the power of images to tell stories and evoke emotions.

I specialize in wedding, portrait, and nature photography, with a style that emphasizes natural light and authentic moments. When I'm not behind the camera, I'm coding and building web applications.

// Example JavaScript for portfolio filtering
function filterPortfolio(category) {
  const items = document.querySelectorAll('.gallery-item');
 
  items.forEach(item => {
    if (category === 'all' || item.dataset.category === category) {
      item.style.display = 'block';
    } else {
      item.style.display = 'none';
    }
  });
}
